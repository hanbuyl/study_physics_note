## 1. 활성화 시간결정의 개념

### 1-1. 기본 아이디어

닫힌계 DTC와 달리, **활성화 시간결정(activated TC)**은 다음 요소로 구성된다:

```
필요 요소:
┌────────────────────────────────────┐
│  1. 이중 우물 포텐셜 (bistability) │
│  2. 소산 (damping)                 │
│  3. 열적 잡음 (thermal noise)      │
│  4. 주기적 구동 (driving)          │
└────────────────────────────────────┘
```

핵심: 잡음이 시스템을 두 우물 사이에서 확률적으로 **전환(switching)** 시키며, 이 전환이 구동과 동기화될 수 있다.

### 1-2. 고전적 모형

이중 우물 포텐셜의 Langevin 방정식:

$$\dot{x} = -\frac{\partial V(x,t)}{\partial x} + \xi(t)$$

$$V(x, t) = -\frac{a(t)}{2}x^2 + \frac{b}{4}x^4$$

여기서 $a(t)$를 주기 $T$로 변조.

---

## 2. 이중 우물 포텐셜 모형

### 2-1. 정적 이중 우물

$a > 0$, $b > 0$이면 이중 우물:

$$V(x) = -\frac{a}{2}x^2 + \frac{b}{4}x^4$$

$$x_{min} = \pm\sqrt{a/b}, \quad V_{barrier} = \frac{a^2}{4b}$$

```
V(x)
  │      ┃           ┃
  │  ────╋───────────╋────
  │      ┃           ┃
  │   ╭──╯           ╰──╮
  │   │   위치 -x₀      │
  └───┼─────────────────┼── x
     -x₀               +x₀
```

두 최솟값 $\pm x_0$ 사이를 잡음이 넘나든다.

### 2-2. Floquet 변조

$a(t)$를 주기 $T$로 변조:

$$a(t) = a_0 + \delta a \cos(2\pi t/T)$$

변조로 인해 두 우물의 상대적 깊이가 교대로 변한다.

## 3. 확률적 공명과 $2T$ 진동

### 3-1. 전환 메커니즘

잡음이 포텐셜 장벽을 넘도록 시스템을 유도:

**Kramers 이탈률:**

$$r \sim e^{-V_{barrier}/k_BT}$$

온도 $T$가 높을수록, 장벽이 낮을수록 전환이 빠름.

### 3-2. 구동과의 동기화

구동이 $T$ 주기로 두 우물을 교대로 낮추면:

```
시간 0:   왼쪽 우물이 더 낮음 → 시스템이 왼쪽으로 이동
시간 T/2: 오른쪽 우물이 더 낮음 → 시스템이 오른쪽으로 이동
시간 T:   다시 왼쪽이 낮음 → ...
```

이것은 구동 주기 $T$와 같은 주기로 진동. **DTC가 아니다!**

### 3-3. 어떻게 $2T$ 진동이 나오는가?

구동이 두 우물을 *같이* 낮추고 높이면 (즉, $|V_{barrier}|$를 변조하면):

```
n이 짝수: 장벽이 낮음 → 잡음이 전환 유도 → 상태 바뀜
n이 홀수: 장벽이 높음 → 잡음이 전환 못 일으킴 → 상태 유지
결과: 매 2주기마다 한 번 전환 → 2T 진동!
```

이것이 활성화 시간결정의 핵심 메커니즘.

## 4. 활성화 시간결정의 특성

### 4-1. 파라미터 조건

DTC가 되려면:
1. **온도 조건**: $k_BT \approx V_{barrier}$ (너무 낮으면 전환 없음, 너무 높으면 무질서)
2. **구동 조건**: 구동이 정확히 $2T$ 전환을 유도
3. **유한 크기 효과**: 단일 입자에서는 유한 수명, 많은 입자 → 열역학적 DTC

### 4-2. 잡음의 역할

```
잡음 너무 약함: 장벽 못 넘음 → 전환 없음 → DTC 없음
잡음 적당함:   동기화된 전환 → DTC 존재  ← 최적 영역
잡음 너무 강함: 무작위 전환 → 동기화 깨짐 → DTC 없음
```

이것은 **확률적 공명(stochastic resonance)** 현상의 변형!

### 4-3. 수명

활성화 TC는 유한한 수명을 가진다:

$$\tau_{DTC} \sim e^{\Delta V / k_BT}$$

충분히 낮은 온도에서 수명이 지수적으로 길어진다.

## 5. 왜 "활성화"인가: Kramers 이론

### 5-1. Kramers 이탈률

열적 활성화(thermal activation)에 의한 장벽 넘기:

$$r_{Kramers} = \frac{\omega_{min}\omega_{max}}{2\pi\gamma} e^{-\Delta V/k_BT}$$

여기서:
- $\omega_{min}$: 최솟값에서의 진동 주파수
- $\omega_{max}$: 최댓값(장벽 꼭대기)에서의 진동 주파수
- $\gamma$: 감쇠 계수
- $\Delta V$: 장벽 높이

### 5-2. "활성화" TC의 의미

활성화 TC는 이 Kramers 활성화 과정을 사용하여 $2T$ 전환을 유도한다는 뜻. 잡음이 없으면 전환이 없고 DTC도 없다.

```
잡음 = TC의 원동력 (연료)
구동 = TC의 방향 (조향)
소산 = TC의 안정화 (제동)
```

## 6. 핵심 통찰 정리

> **잡음이 DTC를 만든다:** 활성화 TC에서 잡음은 DTC의 적이 아니라 *원동력*이다. 잡음이 없으면 장벽을 넘지 못하고 전환이 일어나지 않는다.

> **구동과의 동기화:** 구동이 장벽 높이를 주기적으로 변조하여, 잡음에 의한 전환이 매 2주기마다 일어나도록 동기화한다. 이것이 자발적 $2T$ 진동의 근원이다.

> **유한 수명의 DTC:** 활성화 TC는 진정한 열역학적 극한에서는 유한 수명이다 (Kramers 이탈률). 하지만 $e^{\Delta V/k_BT}$으로 지수 증가하므로 실용적으로 매우 긴 수명 가능.
