## 1. 매개변수 공진(Parametric Resonance)

### 1-1. 기본 개념

매개변수 공진은 시스템의 **파라미터(매개변수)** 를 주기적으로 변조할 때 발생하는 공진이다.

**예시: 그네**

그네의 길이를 주기 $T$로 변화시키면 구동 주파수가 고유 주파수의 2배일 때 진폭이 증가:

$$\ddot{\theta} + \left[\omega_0^2 + \epsilon\cos(2\omega_0 t)\right]\theta = 0$$

이것이 **Mathieu 방정식**. $\epsilon$ 이 작아도 공진이 일어날 수 있다.

### 1-2. 주요 특성

| 보통 공진 | 매개변수 공진 |
|-----------|-------------|
| 구동 주파수 = 고유 주파수 | 구동 주파수 = 2 × 고유 주파수 |
| 위상 제한 있음 | 위상 제한 없음 (모든 초기 위상에서) |
| 선형 시스템 | 파라미터 변조 |

### 1-3. 매개변수 공진과 $2T$ 진동

매개변수 공진에서 진자의 응답 주파수는 $\omega_0 = \omega_{drive}/2$이므로,
주기가 $T_{drive}$면 응답 주기가 $2T_{drive}$.

이것은 $2T$ 진동이다 — DTC처럼 보인다!

## 2. Faraday 파: 공간 패턴의 형성

### 2-1. Faraday 실험 (1831)

수직으로 진동하는 용기 속 유체 표면에 형성되는 패턴.

- 용기 구동 주파수: $2f$
- Faraday 파 주파수: $f$ (절반!)

```
구동:    ↑↓↑↓↑↓↑↓  (주파수 2f)
표면파: ╱╲  ╱╲  ╱╲  (주파수 f, 주기 2T)
```

### 2-2. Faraday 파와 DTC

Faraday 파는 $2T$ 진동을 보인다. 이것이 DTC인가?

**NO.** 이유:

1. **공간 패턴**: Faraday 파는 공간 $+$ 시간 패턴. 순수 시간 질서가 아님
2. **무한 시스템에서 불안정**: 파동이 전파되고 상호 간섭하여 복잡한 패턴 형성
3. **에르고딕성 문제**: 고전 유체는 카오스 → 장시간에서 질서 소멸

## 3. KAM 정리와 적분 가능성

### 3-1. KAM 정리 (Kolmogorov-Arnold-Moser)

적분 가능한 Hamiltonian 시스템에 작은 섭동을 가했을 때:
- 대부분의 초기 조건에서 **불변 토러스(invariant tori)** 가 보존됨
- 카오스가 발생하지 않고 준주기적 운동 유지

```
적분 가능 시스템:      KAM 토러스 유지
                    ────────────────
작은 섭동 후:        대부분의 토러스 생존 (KAM)
                    ← 일부는 카오스 해 (공진 밴드)

큰 섭동 후:          대부분 카오스화 → 에르고딕
```

### 3-2. Floquet 시스템에서의 KAM

주기 구동 시스템에서:
- 구동이 약하면: KAM 토러스 유지 → 에르고딕성 없음 → 일부 초기 조건에서 안정
- 구동이 강하면: 전역 카오스 → 에르고딕성 → DTC 불가

## 4. 고전 시스템이 DTC가 될 수 없는 이유

> **핵심 논지:** 고전적인 비선형 진동자나 유체는 $2T$ 진동을 보일 수 있지만, 이것이 진정한 DTC 질서를 구성하지는 않는다.

**이유 1: 에르고딕성**

상호작용하는 많은 자유도를 가진 고전 시스템은 일반적으로 카오스적이고 에르고딕하다.
장시간에서 초기 조건 정보가 소멸 → $2T$ 진동 소멸.

**이유 2: 강성 부재**

고전 매개변수 공진은 정확한 파라미터(공진 조건)에서만 작동.
파라미터를 조금 바꾸면 공진이 깨짐 → 강성 없음.

**이유 3: 열역학적 불안정성**

$$\text{고전 시스템} \xrightarrow{t\to\infty} \text{열평형(에르고딕)} \xrightarrow{} \text{DTC 불가}$$

그러나! 논문 후반부(Unit 6)에서 소산(dissipation) + 잡음이 있는 열린계에서는 고전 DTC가 가능함을 보인다.

---

## 5. 핵심 통찰 정리

> **매개변수 공진 ≠ DTC:** Faraday 파, 매개변수 공진 등 $2T$ 현상은 흔하지만, DTC의 요건인 강성, 자발성, 열역학적 안정성을 충족하지 못한다.

> **KAM의 역할:** KAM 정리가 보호하는 불변 토러스는 일부 초기 조건에서 에르고딕성을 막아주지만, 많은 자유도의 시스템 전체로는 에르고딕성을 막을 수 없다.

> **고전 → 양자:** 고전계에서 DTC가 어렵다는 것이 양자 MBL이 DTC 연구의 핵심이 된 이유다. 양자 MBL은 에르고딕성을 *전역적으로* 깨뜨릴 수 있다.
