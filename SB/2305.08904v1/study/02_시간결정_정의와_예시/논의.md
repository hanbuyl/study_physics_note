## Q-A. Eq.(3)에서 극한 순서가 왜 중요한가? $N\to\infty$를 먼저, $t\to\infty$를 나중에 취하는 이유는?

### 핵심 논점

두 극한의 순서가 바뀌면 DTC 신호가 사라진다. 이것이 DTC 정의의 핵심적인 수학적 섬세함이다.

---

### 순서가 바뀌면 ($t\to\infty$ 먼저, 유한 $N$에서) 어떻게 되는가

유한한 양자계는 에너지 스펙트럼이 **이산(discrete)** 하다. 이 경우 두 가지 문제가 생긴다.

**문제 1: 재현 (Recurrence)**

Poincaré 재현 정리에 의해, 유한 계는 충분히 오래 기다리면 초기 상태에 임의로 가까이 되돌아온다. 따라서 $2T$ 진동 신호는 소멸했다가 재현하기를 반복하고, 진정한 "장시간 극한"이 정의되지 않는다.

**문제 2: 유한 계에서 "고양이 상태"의 불안정성**

DTC 상태 $|A\rangle = \frac{1}{\sqrt{2}}(|\phi_+\rangle + |\phi_-\rangle)$는 Floquet 고유 상태들의 중첩이다. 유한 계에서는 준에너지 스펙트럼에 다른 많은 준위들이 있고, 이들과의 결합이 $|A\rangle$을 서서히 dephase시킨다.

결과적으로:

$$\lim_{t\to\infty}^{\text{유한 } N} \langle O(nT)\rangle \;\to\; 0$$

시간 평균에서 $(-1)^n$ 항이 상쇄되어 신호가 0이 된다. DTC가 있어도 없는 것처럼 보인다.

---

### 올바른 순서 ($N\to\infty$ 먼저, 그 다음 $t\to\infty$)

열역학적 극한 $N\to\infty$에서 DTC 메커니즘(MBL, BEC 등)이 작동하면:

- 준에너지 $\epsilon_+$와 $\epsilon_-$ 사이의 gap이 안정화됨
- 다른 준위들과의 결합이 억제됨
- $|A\rangle$ 상태가 진정으로 안정화 → $(-1)^n m_0$ 진동이 $t\to\infty$에서도 유지

$$\lim_{t\to\infty}\left[\lim_{N\to\infty} \frac{1}{N}\sum_i\langle\hat{O}_i(nT)\rangle\right] = (-1)^n m_0 \neq 0$$

---

### 평형 SSB와의 완전한 유추

이 극한 순서 문제는 평형계의 자발적 대칭 깨짐에서도 똑같이 나타난다.

**Ising 강자성체에서 자화 $M$:**

$$\lim_{h\to 0}\left[\lim_{N\to\infty} M(h, N)\right] = M_0 \neq 0 \quad \text{(SSB 있음)}$$

$$\lim_{N\to\infty}\left[\lim_{h\to 0} M(h, N)\right] = 0 \quad \text{(SSB 없음)}$$

| | Ising SSB | DTC SSB |
|--|----------|---------|
| 열역학적 극한 | $N\to\infty$ 먼저 | $N\to\infty$ 먼저 |
| 다음 극한 | $h\to 0$ (대칭 깨짐 제거) | $t\to\infty$ (과도 응답 제거) |
| 결과 | 자화 $M_0 \neq 0$ | DTC 진동 $m_0 \neq 0$ |

> **핵심:** 극한 순서는 단순한 수학적 형식이 아니다. 열역학적 극한을 먼저 취한다는 것은 "충분히 큰 시스템에서 DTC 메커니즘이 작동하도록 한 뒤, 그 상태가 장시간 안정한지 묻는다"는 물리적 의미를 담고 있다.

## Q-B. "강성(rigidity)"이 수학적으로 정확히 무슨 의미인가? "위상(phase)"이라는 개념과 어떻게 연결되는가?

### 핵심 논점

강성은 DTC를 단순한 공진과 구별하는 핵심 특성이며, "위상"으로서의 DTC를 정의하는 기반이다.

---

### 강성의 직관적 의미

단순 공진과 DTC의 차이:

```
단순 공진 (예: 기타 줄):
  구동 주파수 바꿈 → 공명 주파수도 바뀜
  파라미터 → 응답이 연속적으로 변화

DTC의 강성:
  구동 파라미터 변화 (예: θ = π → π + ε)
  → 응답 주파수는 여전히 정확히 2T에 "잠겨(locked)" 있음
  → 주파수가 연속적으로 이동하지 않음
```

---

### 수학적 정의: 주파수 잠김(Frequency Locking)

Kicked Ising 모형에서 킥 각도를 $\theta = \pi + \epsilon$으로 변화시킬 때, DTC 신호의 Fourier 변환:

$$\tilde{O}(\omega) = \sum_n \langle\hat{O}(nT)\rangle e^{i\omega nT}$$

**DTC 상 내부 (강성 있음):**

$$\text{peak 위치} = \frac{\pi}{T} \quad \text{for all } |\epsilon| < \epsilon_c$$

파라미터 $\epsilon$이 변해도 peak가 $\omega = \pi/T$에 고정된다.

**DTC 상 바깥 (강성 없음):**

$$\text{peak 위치} = \frac{\pi}{T} + f(\epsilon) \quad \text{(연속적으로 이동)}$$

---

### "위상(Phase)"으로서의 DTC

평형 물질 상태(고체, 강자성체 등)는 특정 파라미터 범위에서 안정한 "상(phase)"을 이룬다.

DTC도 마찬가지로 Floquet 파라미터 공간에서 유한한 영역을 차지하는 하나의 "상"이다.

```
Kicked Ising 위상 다이어그램:

  킥 오차 ε
  ↑
  │    에르고딕 상 (DTC 없음)
  │    ══════════════════════
  │         DTC 상
  │    (ω = π/T에 잠김, 강성 있음)
  │    ══════════════════════
  │    에르고딕 상 (DTC 없음)
  └──────────────────────────→ 무질서 강도 W
```

DTC 상 내부에서는 $\epsilon$을 어떻게 바꿔도 $2T$ 진동이 유지된다. 이 "유한한 면적의 영역"이 존재한다는 것이 강성의 수학적 의미다.

---

### 위상 전이(Phase Transition)와의 연결

DTC 상과 에르고딕 상 사이의 경계 $\epsilon = \epsilon_c$에서 **Floquet 위상 전이**가 일어난다.

- $|\epsilon| < \epsilon_c$: DTC 상, 강성 있음
- $|\epsilon| > \epsilon_c$: 에르고딕 상, DTC 붕괴

이 전이는 평형계의 상전이와 유사하지만, 열역학적 평형이 아닌 Floquet 시스템에서 일어나는 **비평형 위상 전이**다.

> **핵심:** 강성은 "DTC가 파라미터 공간에서 유한한 크기의 위상을 이룬다"는 것을 의미한다. 단 하나의 점(fine-tuned parameter)에서만 작동하는 것이 아니라, 열린 구간에서 안정하다. 이것이 DTC를 단순한 공진과 구별하는 물리적 의미이며, "시간의 결정(crystal)"이라는 이름이 붙은 이유이기도 하다 — 공간 결정도 특정 격자 상수에서만 존재하는 게 아니라 유한한 파라미터 범위에서 안정하다.

## Q-C. AC 조셉슨은 외부 구동이 있어서 자발성이 없다고 했다. 그런데 Kicked Ising도 외부에서 킥을 가하는데, 왜 이쪽은 자발적인가?

### 핵심 논점

"외부 구동이 있다"는 것과 "자발적이지 않다"는 것은 다른 말이다. 핵심은 구동이 **응답의 주파수를 직접 결정하는가**의 여부다.

---

### AC 조셉슨 효과: 구동이 주파수를 강제한다

DC 전압 $V$를 가하면 접합 위상이 일정한 속도로 변한다:

$$\dot{\phi} = \frac{2eV}{\hbar} \quad \Rightarrow \quad \omega_{AC} = \frac{2eV}{\hbar}$$

구동($V$)과 응답 주파수($\omega_{AC}$) 사이의 관계:

$$V \uparrow \quad \Rightarrow \quad \omega_{AC} \uparrow \quad \text{(연속적, 일대일 대응)}$$

즉, 구동이 응답 주파수를 **직접, 연속적으로** 결정한다. "자발성"이 없다.

---

### Kicked Ising DTC: 구동이 주파수를 강제하지 않는다

킥 주기 $T$로 구동하는데 응답 주기는 $2T$. 구동과 응답의 관계:

$$\text{구동 주기 } T \quad \nRightarrow \quad \text{응답 주기 } T$$

오히려 시스템이 *스스로* $2T$를 선택한다. 왜?

$\mathbb{Z}_2$ 대칭이 자발적으로 깨지기 때문이다. 구동 $U_F$는 $|A\rangle \leftrightarrow |B\rangle$ 교환 대칭을 가지고 있다. 즉:

$$U_F: \quad |A\rangle \to |B\rangle, \quad |B\rangle \to |A\rangle$$

이 대칭 아래 구동 자체는 $|A\rangle$과 $|B\rangle$ 중 어느 쪽도 선호하지 않는다. 그런데 열역학적 극한에서 시스템은 둘 중 하나를 자발적으로 선택한다.

---

### 두 경우의 결정적 차이: 위상(Phase) 자유도

동일한 구동 파라미터로 두 개의 시스템을 준비했다고 하자:

**AC 조셉슨 (비자발):**

```
시스템 1: V 인가 → ω = 2eV/ℏ 로 진동
시스템 2: V 인가 → ω = 2eV/ℏ 로 진동 (동일 주파수, 위상도 구동에 의해 결정)
```

**Kicked Ising DTC (자발):**

```
시스템 1: 킥 주기 T → |A⟩에서 시작 → +1, -1, +1, -1 ...
시스템 2: 킥 주기 T → |B⟩에서 시작 → -1, +1, -1, +1 ...
```

두 시스템은 같은 구동을 받지만 **반대 위상**으로 진동할 수 있다. 구동이 위상을 결정하지 못한다. 이것이 자발성의 수학적 표현이다.

---

### 정리: 자발성의 판별 기준

| | AC 조셉슨 | Kicked Ising DTC |
|--|----------|-----------------|
| 구동 주기 | $T_{drive} = 2\pi/\omega_{AC}$ 없음 (DC) | $T$ |
| 응답 주기 | $2\pi/\omega_{AC}$ (구동이 결정) | $2T$ (시스템이 선택) |
| 위상 자유도 | 없음 (구동이 고정) | 있음 (0 또는 $\pi$ 중 선택) |
| 파라미터 변화 시 | 주파수 연속 변화 | 주파수 $2T$에 고정(강성) |
| 자발적? | ❌ | ✓ |

> **핵심:** 자발성의 기준은 "외부 구동이 있는가"가 아니라, "구동이 $2T$ 응답을 직접 강제하는가"다. Kicked Ising의 구동은 $T$ 주기이고, $2T$ 응답은 시스템이 $\mathbb{Z}_2$ 대칭을 자발적으로 깨면서 스스로 선택한다. AC 조셉슨은 응답 주파수가 외부 전압에 의해 일대일로 결정된다.

## Q-D. Kicked Ising 모형에서 킥 각도 $\theta$가 $\pi$에서 크게 벗어나면 DTC가 왜 붕괴하는가?

### 핵심 논점

MBL이 작은 오차 $\epsilon$을 흡수할 수 있는 데는 한계가 있다. $\theta$가 $\pi$에서 충분히 벗어나면 DTC 상에서 에르고딕 상으로 위상 전이가 일어난다.

---

### 완벽한 킥 ($\theta = \pi$): 왜 자명하게 DTC인가

$$U_{kick}(\pi) = e^{-i\pi\sum_i\sigma_i^x} = \prod_i (-i\sigma_i^x)$$

이것은 모든 스핀을 정확히 $\pi$ 회전 → 완전히 뒤집기.

초기 상태 $|\uparrow\uparrow\cdots\rangle$에서:

```
n=0:  |↑↑↑↑⟩
n=1:  |↓↓↓↓⟩  (정확히 flip)
n=2:  |↑↑↑↑⟩  ...
```

$2T$ 진동. 그러나 이것은 파라미터가 정확히 $\pi$일 때만이라 자명하고 강성이 없다.

---

### 불완전한 킥 ($\theta = \pi + \epsilon$): MBL의 보호

오차 $\epsilon$이 생기면 킥이 완전한 flip이 아니게 된다. 그런데 MBL이 있으면 DTC가 유지된다. 왜?

MBL의 l-bit 언어로 보면, 유효 Floquet Hamiltonian은:

$$H_F \approx \sum_i \tilde{h}_i\tau_i^z + \sum_{ij}J_{ij}\tau_i^z\tau_j^z + \cdots$$

킥 오차 $\epsilon$은 이 Hamiltonian에 작은 횡자기장(transverse field) 항을 추가한다:

$$\delta H \sim \epsilon \sum_i \tau_i^x$$

MBL의 l-bit 구조에서 이 횡자기장의 효과는 **국소적(local)** 으로만 남는다. 즉, 스핀 $i$의 오차가 인접한 스핀 $j$로 전파되지 않는다. 결과적으로 각 스핀이 독립적으로 $2T$ 진동을 유지하고, 공간 평균 $\frac{1}{N}\sum_i \langle\sigma_i^z\rangle$는 여전히 $(-1)^n m_0$로 진동한다.

---

### 큰 오차 ($|\epsilon|$ 크거나, $\theta \approx 0$ 또는 $\theta \approx \pi/2$): 보호 실패

$\epsilon$이 임계값 $\epsilon_c$를 넘으면:

**이유 1: l-bit 보호의 붕괴**

횡자기장 $\epsilon\tau_i^x$의 크기가 MBL의 국소화 에너지 스케일($\sim J_{ij}$)과 비슷해지면 l-bit 구조 자체가 흔들린다. 국소적 보호가 실패하면서 오차가 공간적으로 퍼진다.

**이유 2: 운동학적 한계**

$\theta = \pi/2$에 가까워지면 킥 자체가 flip이 아니라 $90°$ 회전이 된다. 이때 두 번 적용해도 원래 상태로 돌아오지 않는다:

```
θ = π:    2번 → 원래 상태 복귀 (2T 주기)
θ = π/2:  4번 → 원래 상태 복귀 (4T 주기) ... DTC가 아님
θ = 0:    flip 없음 → 진동 없음
```

---

### 위상 다이어그램으로 정리

```
킥 각도 θ
  π ─────────────┬─────────────
                 │  DTC 상
                 │  (MBL이 보호)
π+εc ────────────┼──────────── ← 위상 전이
                 │  에르고딕 상
                 │  (DTC 붕괴, 가열)
  0 ─────────────┴─────────────
                      무질서 강도 W →
```

- $|\theta - \pi| < \epsilon_c$: MBL이 오차를 흡수, DTC 유지
- $|\theta - \pi| > \epsilon_c$: MBL 보호 실패, 에르고딕 상으로 전이

위상 전이 경계 $\epsilon_c$는 무질서 강도 $W$가 클수록 더 넓다. 강한 MBL일수록 더 큰 오차를 허용한다.

> **핵심:** MBL의 보호 능력에는 한계가 있다. $\theta$가 $\pi$에서 멀어지면 킥 오차의 규모가 MBL의 국소화 에너지 스케일을 초과하고, l-bit 보호가 붕괴된다. 이때 에르고딕성이 회복되면서 Floquet 가열이 시작되고 DTC 신호가 소멸한다. 이 경계가 바로 DTC↔에르고딕 Floquet 위상 전이다.
