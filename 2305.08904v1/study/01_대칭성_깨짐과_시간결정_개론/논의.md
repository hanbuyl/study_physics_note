## Q1. 연속 시간 대칭은 관측량의 시간 의존성을 만들지 않는다는 건 알겠다. 그런데 왜 이산 시간결정에서는 가능해지지? 똑같은 time-evolution formalism으로 작성되지 않나?

### 핵심 논점

형식론(formalism)은 같다. 그러나 "무엇이 자발적으로 깨지는가"의 구조가 다르다.

---

### 연속 시간 독립 Hamiltonian: 왜 안 되는가

에너지 고유 상태의 시간 진화:

$$|\psi(t)\rangle = e^{-iEt/\hbar}|E\rangle$$

관측량의 기댓값:

$$\langle O\rangle(t) = \langle E|e^{+iEt/\hbar}\hat{O}\,e^{-iEt/\hbar}|E\rangle = \langle E|\hat{O}|E\rangle = \text{const}$$

위상 인자가 정확히 상쇄 → 상수.

**반론: 에너지 고유 상태의 중첩이면?**

$$|\psi\rangle = \frac{1}{\sqrt{2}}(|E_0\rangle + |E_1\rangle) \quad \Rightarrow \quad \langle O\rangle(t) \propto \cos\!\left(\frac{E_1 - E_0}{\hbar}t\right)$$

이건 진짜로 시간에 따라 진동한다. 그러나 시간결정이 아닌 이유:

| 판별 기준 | 이 중첩 상태 |
|-----------|-------------|
| 자발적(Spontaneous)? | ❌ 초기 조건을 정확히 조절해야만 이 중첩이 생김 |
| 강성(Rigidity)? | ❌ 진동 주파수 $(E_1-E_0)/\hbar$는 파라미터에 민감 |
| 열역학적 극한? | ❌ $N\to\infty$에서 이런 "고양이 상태" 중첩은 붕괴 |

Watanabe-Oshikawa 정리는 이것을 엄밀히 말한 것: **평형 상태(ground state 또는 열적 앙상블)에서는** 어떤 초기 조건을 고르든 이런 진동이 자발적으로, 강인하게, 열역학적 극한에서 존재할 수 없다.

---

### Floquet 시스템: 왜 가능해지는가

가장 단순한 DTC 상황. $U_F$가 다음을 만족하는 두 상태가 있다면:

$$U_F|A\rangle = |B\rangle, \qquad U_F|B\rangle = |A\rangle$$

$U_F$의 실제 고유 상태(quasienergy eigenstates):

$$|\phi_+\rangle = \frac{|A\rangle + |B\rangle}{\sqrt{2}}, \quad \epsilon_+ = 0$$

$$|\phi_-\rangle = \frac{|A\rangle - |B\rangle}{\sqrt{2}}, \quad \epsilon_- = \frac{\pi}{T}$$

초기 상태 $|\psi_0\rangle = |A\rangle$에서:

$$\langle O\rangle(nT) = \underbrace{\frac{\langle\phi_+|O|\phi_+\rangle + \langle\phi_-|O|\phi_-\rangle}{2}}_{\text{상수}} + \underbrace{\langle\phi_+|O|\phi_-\rangle \cdot (-1)^n}_{2T \text{ 진동}}$$

**연속 vs Floquet의 근본 차이:**

```
연속 Hamiltonian의 고유 상태 중첩:
  위상 e^{i(E_m - E_n)t/ℏ} → 어떤 주파수도 가능, fine-tuned

Floquet 고유 상태 중첩:
  위상 e^{i(ε₊ - ε₋)nT/ℏ} = e^{iπn} = (-1)^n → 항상 정확히 2T
```

구동이 준에너지(quasienergy) 차이를 $\pi/T$로 고정시켜 준다. 이것이 핵심.

---

### SSB와의 연결: 왜 "자발적"인가

연속 경우의 문제: "왜 하필 그 주파수인가?"를 설명할 구조가 없다.

Floquet 경우: 구동 자체가 이산 시간 대칭 $\mathbb{Z}_T$를 먼저 제공한다. DTC는 거기서:

```
구동이 제공:   이산 대칭 ℤ_T  (t → t + T)
DTC가 깨뜨림:  ℤ_T → ℤ_{2T}  (t → t + 2T로만 불변)
```

이것은 Ising 강자성체가 $\mathbb{Z}_2$ 대칭을 자발적으로 깨는 것과 완전히 동일한 구조.
Ising에서도 Hamiltonian은 대칭인데 바닥 상태가 $|\uparrow\uparrow\cdots\rangle$ 또는 $|\downarrow\downarrow\cdots\rangle$ 중 하나를 선택한다.

> **한 줄 요약:** 형식론은 동일하지만, 연속 경우에는 "어떤 주파수를 깨뜨릴 것인가"를 결정해줄 구조 자체가 없다. Floquet 구동이 $\mathbb{Z}_T$라는 이산 구조를 먼저 만들어주기 때문에, 거기서 $\mathbb{Z}_2$ 대칭 깨짐(짝수/홀수 스텝 구별)이 일어날 수 있다.

## Q2. 왜 갑자기 ε₊, ε₋가 나왔는지 모르겠어.

### 핵심 논점

Q1에서 $\epsilon_\pm$를 설명 없이 쓴 것이 문제였다. 순서대로 유도한다.

---

### 출발점: Floquet 시스템에서 뭘 대각화하는가

시간 독립 Hamiltonian → $H$를 대각화:

$$H|n\rangle = E_n|n\rangle \quad \Rightarrow \quad \text{시간 진화 } e^{-iE_n t/\hbar}$$

Floquet 시스템($H(t) = H(t+T)$) → $H$ 자체는 시간에 따라 변하므로 대각화 불가. 대신 **한 주기 진화 연산자 $U_F$를 대각화**:

$$U_F = \mathcal{T}\exp\!\left(-\frac{i}{\hbar}\int_0^T H(t)\,dt\right)$$

$U_F$는 유니터리 행렬 → 고유값이 반드시 위상 인자:

$$U_F|\phi_n\rangle = e^{-i\epsilon_n T/\hbar}|\phi_n\rangle$$

$\epsilon_n$ = **준에너지(quasienergy)**, $|\phi_n\rangle$ = **Floquet 고유 상태**.
에너지 고유 상태의 Floquet 버전이다.

---

### DTC 조건에서 ε±가 어디서 나오는가

가정: $|A\rangle$, $|B\rangle$ 두 상태가 DTC 조건을 만족:

$$U_F|A\rangle = |B\rangle, \qquad U_F|B\rangle = |A\rangle$$

$\{|A\rangle, |B\rangle\}$ 기저의 행렬로 표현:

$$U_F = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}$$

이 행렬을 대각화(선형대수):

$$\text{고유값: } \lambda = +1 \text{ 또는 } -1$$

$$|\phi_+\rangle = \frac{|A\rangle + |B\rangle}{\sqrt{2}}, \quad U_F|\phi_+\rangle = +1\cdot|\phi_+\rangle$$

$$|\phi_-\rangle = \frac{|A\rangle - |B\rangle}{\sqrt{2}}, \quad U_F|\phi_-\rangle = -1\cdot|\phi_-\rangle$$

고유값 $e^{-i\epsilon T/\hbar}$에 대입:

$$+1 = e^{-i\epsilon_+ T/\hbar} \;\Rightarrow\; \epsilon_+ = 0$$

$$-1 = e^{-i\epsilon_- T/\hbar} = e^{-i\pi} \;\Rightarrow\; \epsilon_- = \frac{\pi}{T}$$

즉, $\epsilon_\pm$는 임의로 설정한 것이 아니라 **"한 번 킥하면 뒤집힌다"는 DTC 조건을 $U_F$ 행렬로 쓰고 대각화하면 자동으로 나오는 결과**다.

---

### 왜 $(-1)^n$이 $2T$ 진동을 만드는가

$|A\rangle = \frac{1}{\sqrt{2}}(|\phi_+\rangle + |\phi_-\rangle)$이므로 $n$번 킥 후:

$$U_F^n|A\rangle = \frac{1}{\sqrt{2}}\!\left((+1)^n|\phi_+\rangle + (-1)^n|\phi_-\rangle\right)$$

```
n = 0 (짝수):  (+|φ₊⟩ + |φ₋⟩)/√2 = |A⟩
n = 1 (홀수):  (+|φ₊⟩ - |φ₋⟩)/√2 = |B⟩
n = 2 (짝수):  (+|φ₊⟩ + |φ₋⟩)/√2 = |A⟩
...
```

$(-1)^n$ 인자가 $2T$ 진동의 원천이고, 이것은 $\epsilon_- = \pi/T$에서 직접 나온다.

> **유도 순서 정리:**
> DTC 조건 → $U_F$ 행렬 표현 → 대각화 → 고유값 ±1 → 준에너지 $0$과 $\pi/T$
> → $(-1)^n$ 인자 → $2T$ 진동
