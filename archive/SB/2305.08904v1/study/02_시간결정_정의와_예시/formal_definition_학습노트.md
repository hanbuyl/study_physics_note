## 1. DTC의 공식 정의: Eq.(3)

논문은 DTC를 다음과 같이 정의한다.

주기 $T$로 구동되는 Floquet 시스템에서, 어떤 국소 관측량 $\hat{O}_i$ (사이트 $i$에서)에 대해:

$$\lim_{N\to\infty} \lim_{t\to\infty} \frac{1}{N} \sum_i \langle \hat{O}_i(t) \rangle \neq 0 \quad \text{for } t = (2n+1)T$$

즉:

| 조건 | 의미 |
|------|------|
| $N \to \infty$ 먼저 | 열역학적 극한 (유한 크기 효과 제거) |
| $t \to \infty$ 나중 | 장시간 안정성 (과도 응답 제거) |
| $2T$ 주기 신호 | 구동 주기의 2배로 진동 |

**극한 순서가 중요하다.** $t \to \infty$ 를 먼저 취하면 유한계에서는 항상 0이 된다.

## 2. 에르고딕성(Ergodicity)이란

> **에르고딕 가설:** 충분히 오래 기다리면 시스템은 에너지 제약과 일치하는 **모든 미시 상태**를 동등하게 방문한다.

에르고딕 시스템의 결과:
- 시간 평균 = 앙상블 평균
- 초기 조건과 무관하게 열평형으로 수렴
- **장기적으로 모든 구별이 사라짐**

```
에르고딕 시스템:
초기 상태 A ─┐
초기 상태 B ─┤─→  t→∞  →  동일한 열평형 상태
초기 상태 C ─┘
```

Floquet 시스템에서의 에르고딕성:
- 충분히 오래 구동하면 시스템이 **무한 온도 상태**로 가열됨(heating)
- 결국 모든 정보가 지워지고 완전히 무질서한 상태

---

## 3. 에르고딕성 깨짐과 DTC의 관계

**에르고딕계**: 시간이 지나면 초기 조건 기억이 소멸 → DTC 불가능

**비에르고딕계**: 초기 조건의 일부 정보가 영원히 보존 → DTC 가능

```
에르고딕성 깨짐의 메커니즘:
┌─────────────────────────────────┐
│  MBL (다체 국소화)              │  ← Unit 4
│  Prethermal plateau             │  ← Unit 5
│  BEC (보즈-아인슈타인 응축)     │  ← Unit 5
│  소산(dissipation) + 잡음      │  ← Unit 6
└─────────────────────────────────┘
모두 에르고딕성을 깨뜨려 DTC를 안정화하는 방법들
```

## 4. 진짜 DTC vs. 아닌 것 구별

**DTC가 아닌 경우들:**

| 현상 | 왜 DTC가 아닌가 |
|------|----------------|
| 단순 공진 | 파라미터 정확히 맞아야 함, 강성 없음 |
| 과도 진동 (transient) | $t \to \infty$ 극한에서 사라짐 |
| 구동과 같은 주기 $T$ 반응 | $2T$ 대칭 깨짐 없음 |
| 유한 크기 효과 | $N \to \infty$ 극한에서 사라짐 |

**진짜 DTC의 특성:**

$$\langle \hat{O}(nT) \rangle \approx (-1)^n \cdot \bar{O} \quad (n \to \infty, N \to \infty)$$

즉 짝수/홀수 스텝에서 교대하는 값이 **열역학적 극한에서도 유한하게 남는다**.

## 5. DTC 진단 방법

### 5-1. 시간 영역 측정

스핀(또는 다른 질서 매개변수)을 $nT$ 마다 측정:

```
⟨S_z(nT)⟩
  +1 │  ●     ●     ●
     │    ●     ●     ●   → 주기 2T 진동
  -1 └─────────────────── n
       0  1  2  3  4  5
```

### 5-2. 주파수 영역 측정

시간 신호의 Fourier 변환에서 $\omega = \pi/T$ (즉, $f = 1/(2T)$)에 peak:

$$\tilde{O}(\omega) = \sum_n \langle \hat{O}(nT) \rangle e^{i\omega nT}$$

DTC면 $\omega = \pi/T$에 안정된 peak 존재.

### 5-3. 강성 검증

구동 파라미터 (예: 펄스 각도 $\theta = \pi + \epsilon$)를 변화시키면서 peak 위치 추적:

```
파라미터 오차 ε →
DTC:      peak가 ω = π/T에 고정 (locked)
비DTC:    peak가 ε에 따라 이동 (unlocked)
```

## 6. 핵심 통찰 정리

> **DTC의 핵심:** 열역학적 극한과 장시간 극한 두 가지를 동시에 충족하는 $2T$ 주기 진동. 단순한 진동이나 공진과의 차이는 *강성(rigidity)*과 *자발성(spontaneity)*에 있다.

> **에르고딕성과의 연결:** 에르고딕 시스템은 결국 모든 기억을 잊기 때문에 DTC가 불가능하다. DTC는 반드시 에르고딕성이 깨진 시스템에서만 가능하다.

> **논문의 분류:** 이 논문은 에르고딕성을 깨는 메커니즘에 따라 DTC를 분류한다 — MBL, Prethermal, BEC, 소산계 등.
