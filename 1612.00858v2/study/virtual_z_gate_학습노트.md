## 1. 트랜스몬 큐비트의 물리적 실체

### 1-1. 조셉슨 접합 (Josephson Junction)

트랜스몬의 핵심 소자는 **조셉슨 접합**이다.

```
    알루미늄 (초전도체)
    ─────────────────────
         Al₂O₃ (~1-2 nm 절연층)
    ─────────────────────
    알루미늄 (초전도체)
```

두 초전도체 사이에 극도로 얇은 절연막을 끼운 샌드위치 구조.
여기에 큰 션트 커패시터 C를 병렬 연결하면 트랜스몬이 된다.

```
        C (션트 커패시터)
    ┌───┤├───┐
    │        │
    └──[JJ]──┘
      조셉슨 접합
```

### 1-2. 거시적 양자 응축 상태 (Macroscopic Quantum Condensate)

초전도 상태에서 전자들은 **쿠퍼 쌍(Cooper pair)** 을 형성한다.

- 쿠퍼 쌍은 스핀 0의 **보존(boson)**
- 보존이므로 보즈-아인슈타인 응축처럼 동일한 바닥 상태로 응축
- 결과: 수십억 개의 쿠퍼 쌍이 **단 하나의 거시적 파동함수**로 기술됨

```
Ψ = |Ψ| · e^{iφ}
```

| 계 | 양자 자유도 |
|----|------------|
| 일반 금속 | 전자 각각이 다른 양자 상태 → 고전적 행동 |
| 초전도체 | 쿠퍼 쌍 전체가 단일 파동함수 → 거시적 양자 행동 |

### 1-3. 큐비트 양자 변수

조셉슨 접합에는 두 켤레(conjugate) 변수가 있다:

| 변수 | 의미 |
|------|------|
| `φ̂` | 접합 양쪽의 초전도 위상 차이 |
| `n̂` | 접합을 건너간 쿠퍼 쌍의 수 |

이 둘의 관계는 위치-운동량과 동일:

```
[φ̂, n̂] = i    (cf. [x̂, p̂] = iℏ)
```

### 1-4. 트랜스몬 Hamiltonian

```
H = 4E_C · n̂²  -  E_J · cos(φ̂)
    ──────────    ───────────────
    충전 에너지    조셉슨 터널링 에너지
    (운동 에너지)  (비선형 포텐셜)
```

이것은 **비선형 진동자(anharmonic oscillator)** 다.

- 조화 진동자: 포텐셜 ∝ φ² → 준위 간격이 모두 동일 → 특정 준위만 구동 불가
- 트랜스몬: 포텐셜 = -E_J·cos(φ̂) → **비선형** → 준위 간격이 불균등

```
에너지
  │
  │  ─────  |2⟩   ω₁₂ = ω₀₁ + α  (α ≈ -235 MHz, 비조화성)
  │
  │  ─────  |1⟩   ω₀₁ ≈ 5 GHz
  │
  │  ─────  |0⟩
```

비조화성(anharmonicity) α 덕분에 ω₀₁ ≠ ω₁₂이므로,
ω₀₁에 공진시킨 드라이브는 |0⟩↔|1⟩만 선택적으로 구동한다.

### 1-5. |0⟩과 |1⟩의 물리적 의미

```
|0⟩ : 비선형 진동자의 바닥 상태
      → 쿠퍼 쌍 집단 진동이 "최소"인 거시적 양자 상태

|1⟩ : 첫 번째 들뜬 상태
      → 쿠퍼 쌍 집단 진동이 양자 하나만큼 "들뜬" 거시적 양자 상태
```

**핵심:** 이것은 단일 전자의 상태가 아니다.
수십억 개의 쿠퍼 쌍이 집단적으로(collectively) 이루는 양자 상태다.

## 2. 왜 마이크로파 드라이브가 σ_X와 결합하는가

### 2-1. 결합의 종류: 용량성(Capacitive) 결합

드라이브 전압이 게이트 커패시터 C_g를 통해 트랜스몬에 전달된다.

```
드라이브 선  V_d(t)
     │
    C_g
     │
  트랜스몬 섬 (island)
```

이것은 전기 쌍극자 상호작용과 동일한 구조:

```
H_drive = -Q · V = -2e · (C_g/C_Σ) · V_d(t) · n̂
                                              ───
                                        전하 연산자
```

| 시스템 | 결합 형태 | 연산자 |
|--------|-----------|--------|
| 원자 (광학) | 전기 쌍극자 `-d·E` | `x̂ ∝ σ_X` |
| NMR (스핀) | 자기 쌍극자 `-μ·B_RF` | `σ_X 또는 σ_Y` |
| 트랜스몬 | 용량성 결합 `-Q·V` | `n̂ ∝ σ_X` |

### 2-2. 왜 n̂ → σ_X 인가?

n̂의 행렬 요소를 에너지 고유 상태 기저로 계산:

```
⟨0|n̂|0⟩ = 0    (대각 요소 없음)
⟨1|n̂|1⟩ = 0

⟨0|n̂|1⟩ = n₀₁ ≠ 0    (비대각 요소만 존재)
```

n̂은 조화 진동자의 위치 연산자 x̂ ∝ (â + â†)에 대응한다.
위치 연산자는 인접 준위만 연결하므로 대각 요소가 없다.

따라서 2준위 근사에서:

```
n̂ ≈ n₀₁ · (|0⟩⟨1| + |1⟩⟨0|) = n₀₁ · σ_X
```

- `σ_Z = |0⟩⟨0| - |1⟩⟨1|` → 에너지를 "측정"하는 연산자 (대각)
- `σ_X = |0⟩⟨1| + |1⟩⟨0|` → 상태를 "연결(전이)"하는 연산자 (비대각)

마이크로파는 에너지를 재는 게 아니라 **전이를 유도**하므로 σ_X 결합이다.

### 2-3. 최종 Lab Frame Hamiltonian

```
H/ℏ = (ω₀₁/2) · σ_Z  +  Ω(t) · cos(ω_D·t - γ) · σ_X
       ─────────────     ───────────────────────────────
       큐비트 에너지      드라이브 (용량성 결합)
```

## 3. Rotating Frame 변환과 RWA 유도

### 3-1. 변환 유니터리

ω_D 속도로 도는 좌표계로 이동:

```
U_rot(t) = e^{i · ω_D · t · σ_Z / 2}
```

새 Hamiltonian:

```
H̃ = U_rot · H · U_rot†  +  iℏ · (∂_t U_rot) · U_rot†
```

### 3-2. 핵심 항등식

```
e^{i·θ·σ_Z/2} · σ_X · e^{-i·θ·σ_Z/2}
```

행렬로 직접 계산 (θ = ω_D·t):

```
= [[0,       e^{iθ} ]]
  [[e^{-iθ}, 0      ]]

= cos(ω_D·t) · σ_X  -  sin(ω_D·t) · σ_Y
```

### 3-3. 곱셈 전개 후 RWA 적용

드라이브 항을 변환하면:

```
σ_X 계수 = cos(ω_D·t - γ) · cos(ω_D·t)
         = [cos(2ω_D·t - γ) + cos(γ)] / 2
            ─────────────────   ───────
             빠른 진동 (버림)    DC 항 (보존)

σ_Y 계수 = [-sin(2ω_D·t - γ) + sin(γ)] / 2
            ─────────────────   ───────
             빠른 진동 (버림)    DC 항 (보존)
```

**회전파 근사(RWA):** 펄스 시간(~ns) 동안 2ω_D(~10 GHz) 항은 수천 번 진동하여 평균 0이 됨. 제거.

### 3-4. 최종 Rotating Frame Hamiltonian

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  H̃/ℏ = (Δ/2) · σ_Z  +  (Ω(t)/2) · [cos(γ)·σ_X + sin(γ)·σ_Y] │
│                                                              │
│         Δ = ω₀₁ - ω_D  (공진 이탈량, detuning)             │
│         Ω(t) : 드라이브 세기 (회전 속도 결정)               │
│         γ : 드라이브 위상 (회전 축 결정)                    │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 4. 왜 XY 회전인가 / 왜 Z 회전은 다른가

### 4-1. XY 회전 (공진, Δ = 0)

Δ = 0이면 σ_Z 항이 사라진다:

```
H̃/ℏ = (Ω(t)/2) · [cos(γ) · σ_X + sin(γ) · σ_Y]
```

이것은 단위 벡터 **n̂ = cos(γ)x̂ + sin(γ)ŷ** 를 축으로 하는 회전 Hamiltonian.
시간 T 후 유니터리 진화:

```
U(T) = exp(-i · θ/2 · [cos(γ) · σ_X + sin(γ) · σ_Y])
```

여기서 θ = Ω·T. 이것은 XY 평면에서 γ 방향 축을 중심으로 θ만큼 회전.

```
γ = 0    → σ_X 회전 (X 게이트)   ✓
γ = π/2  → σ_Y 회전 (Y 게이트)   ✓
γ = φ    → XY 평면에서 φ 방향 축 회전  ✓
```

### 4-2. Z 회전은 왜 다른가?

**Z 회전의 정체:** 중첩 상태에서 |0⟩과 |1⟩의 **상대 위상**을 바꾸는 것.

```
Z(φ) 적용:
α|0⟩ + β|1⟩  →  α|0⟩ + e^{iφ} · β|1⟩
```

이 상대 위상은 |0⟩과 |1⟩이 다른 에너지를 가져 **서로 다른 속도로 위상이 쌓일 때** 자연 발생:

```
|0⟩ → e^{-iE₀t/ℏ} |0⟩
|1⟩ → e^{-iE₁t/ℏ} |1⟩
→ 상대 위상 = e^{-iω₀₁·t}  ← Z 회전
```

그런데 회전 좌표계는 ω₀₁ 속도로 이미 따라 돌기 때문에 이 자연 Z 회전이 제거된다.
**회전 좌표계에서 추가 Z 회전을 만들려면 Δ ≠ 0이 필요하다.**

```
Rotating Frame Hamiltonian의 (Δ/2)·σ_Z 항이 바로 Z 회전의 원천
→ Δ = 0이면 Z 회전 없음
→ Δ ≠ 0이면 Z 회전 발생 (하지만 시간 소요, 오류 발생)
```

### 4-3. 요약

| | 원인 | 시간 | 오류 |
|-|------|------|------|
| **XY 회전** | 공진 마이크로파 (전이 구동) | 유한 (~ns) | 있음 |
| **Z 회전 (물리적)** | 이탈(Δ)로 상대 위상 축적 | 유한 (~ns) | 있음 |
| **Z 회전 (Virtual)** | γ 위상 업데이트 (소프트웨어) | 0 | 없음 |

## 5. Virtual Z-Gate의 원리

### 5-1. 핵심 아이디어

**Z 회전이 관측에 미치는 영향은 오직 이후 X/Y 게이트를 통해서만 나타난다.**
→ 큐비트 상태를 직접 Z 회전시키는 것과, 이후 드라이브의 γ를 바꾸는 것이 관측상 동일하다.

수학적 동치 (논문 Eq. 10-12):

두 펄스에서 두 번째 펄스의 위상을 φ만큼 이동시키면:

```
[두 번째 펄스, γ=φ로 이동] · [첫 번째 펄스, γ=0]
= Z_{-φ} · X_θ · Z_φ · X_θ
```

Z_{-φ}는 Z 측정 결과에 영향을 주지 않는 "앞쪽 위상"이므로,
결과적으로 X_θ · Z_φ · X_θ와 관측상 동일하다.

### 5-2. 구현: phase_register

```python
# 소프트웨어 변수
phase_register = 0.0

# VZ(φ) 실행 → 큐비트에 아무것도 안 함
def virtual_z(phi):
    phase_register += phi    # 숫자 하나만 업데이트
    # time = 0 ns, error = 0

# X_{π/2} 실행 → phase_register 반영
def x_half_pulse():
    gamma = 0 + phase_register    # 기본값에 누적 위상 더함
    send_microwave(amplitude=Ω_G, phase=gamma)
    # 실제로 나가는 펄스는 gamma 방향 축으로 회전
```

### 5-3. 시나리오: H 게이트 실행

H = X_{π/2} · Z_{π/2} · X_{π/2} 분해 기준:

```
초기:  |0⟩, phase_register = 0

────────────────────────────────────────────────
t = 0 → 13.3 ns:  X_{π/2} 물리 펄스
  AWG: γ = 0 + 0 = 0  → X 축 회전
  상태: |0⟩ → (|0⟩ - i|1⟩)/√2

────────────────────────────────────────────────
t = 13.3 ns:  Z_{π/2} → Virtual!
  소프트웨어: phase_register += π/2  → π/2
  큐비트: 변화 없음
  소요 시간: 0 ns

────────────────────────────────────────────────
t = 13.3 → 26.6 ns:  X_{π/2} 물리 펄스
  AWG: γ = 0 + π/2 = π/2  → Y 축 회전
  상태: H|0⟩ (글로벌 위상 제외)

────────────────────────────────────────────────
결과:  26.6 ns 소요 (물리 펄스 2개만)
       기존 방식: ~40 ns (물리 Z 포함 3개)
```

### 5-4. 왜 Z 측정에 영향이 없는가?

Z 측정은 |0⟩과 |1⟩의 **확률(|α|², |β|²)** 만 읽는다.

```
α|0⟩ + β|1⟩  →  P(0) = |α|², P(1) = |β|²
```

Z 회전은 상대 위상 e^{iφ}를 바꾸지만, 확률은 바꾸지 않는다.
따라서 Z 회전이 실제로 큐비트에 일어났는지, 드라이브 γ가 바뀌었는지를
**Z 측정만으로는 구별할 수 없다.**

이것이 Virtual Z-gate가 성립하는 근본 이유다.

### 5-5. 실험 결과 (논문)

| 방법 | EPC (Error per Clifford) |
|------|--------------------------|
| XY_{π/2} 기저셋 | 5.6(1) × 10⁻⁴ |
| HZ 기저셋 (VZ 활용) | 3.0(1) × 10⁻⁴ |
| S 게이트 오류 (VZ 자체) | ≈ 0 (−1.7(1.0) × 10⁻⁵) |

VZ gate는 오류가 사실상 0이며, HZ 기저셋의 EPC가 약 46% 낮다.

## 6. 전체 개념의 연결

```
초전도체의 물리
      │
      │  쿠퍼 쌍이 보존(boson)으로서
      │  거시적 양자 응축 상태를 형성
      ↓
트랜스몬 = 비선형 진동자
      │
      │  조셉슨 접합의 cos(φ) 비선형성이
      │  에너지 준위 간격을 불균등하게 만듦
      ↓
|0⟩, |1⟩ = 이 진동자의 바닥/들뜬 상태
      │
      │  수십억 쿠퍼 쌍의 집단 상태
      │  하나의 artificial atom처럼 동작
      ↓
마이크로파 드라이브
      │
      │  용량성 결합: H_drive = -Q·V ∝ n̂·V_d(t)
      │  n̂의 비대각 구조: n̂ → n₀₁·σ_X
      ↓
Lab Frame:  H = (ω₀₁/2)σ_Z + Ω(t)cos(ω_D·t - γ)·σ_X
      │
      │  회전 좌표계 변환 + RWA
      ↓
Rotating Frame:  H̃ = (Δ/2)σ_Z + (Ω/2)[cos(γ)σ_X + sin(γ)σ_Y]
      │
      ├─ Δ = 0 (공진): XY 평면의 γ 방향 축으로 회전
      │                → X/Y 게이트 구현
      │
      └─ Δ ≠ 0 (이탈): σ_Z 항 → Z 회전 (물리적, 느리고 오류 있음)
                                           │
                                           │  대안
                                           ↓
                             Virtual Z-Gate
                                           │
                             phase_register += φ
                             이후 모든 펄스의 γ에 φ 누적
                                           │
                             Z 회전이 이후 X/Y 게이트를 통해
                             관측에 동일하게 반영됨
                             (Z 측정 불변성에 의해)
```

## 최종 이해 정리

> **수십억 개의 전자들이 쿠퍼 쌍으로 응축한 거시적 양자 상태가 큐비트의 실체다.**
> 이 집단 상태의 바닥/들뜬 상태가 |0⟩/|1⟩이고, 그 중첩을 외부 전압으로 조절한다.
>
> 전압 드라이브의 위상 γ는 Bloch 구 XY 평면에서 회전축의 방향을 결정한다.
> Z축은 에너지 고유 방향(측정 기준축)이므로, Z 회전은 물리적으로 이탈(Δ)이 필요하다.
>
> 그러나 Z 회전이 관측에 영향을 미치는 경로는 **이후 X/Y 게이트** 뿐이다.
> 따라서 큐비트를 직접 건드리는 대신, 소프트웨어의 phase_register에 Z 회전량을 저장해두고
> 이후 물리 펄스를 가할 때 γ에 반영하면, 관측상 완전히 동일한 결과를 얻는다.
> 이것이 **Virtual Z-gate**이며, 시간 0, 오류 0의 완벽한 게이트다.
